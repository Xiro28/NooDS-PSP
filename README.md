# NooDS-PSP
 A Nintendo DS emulator for PSP.
 Based on NooDS: https://github.com/Hydr8gon/NooDS

# What you need:
- Arm9 Bios
- Arm7 Bios
- NDS firmware
- Media Engine plugin (mediaengine.prx)

Then rename them as follows:
  - Arm9 bios -> bios9.bin
  - Arm7 bios -> bios7.bin
  - Firmware  -> firmware.bin
